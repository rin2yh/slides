# slides

[Marp](https://marp.app/) で作成したスライド集です。

## セットアップ

```bash
mise install
npm install
```

## コマンド

| コマンド | 説明 |
|---|---|
| `npm run preview` | プレビューを起動 |
| `npm run build` | HTML を `dist/` に出力 |
| `npm run pdf` | PDF を `dist/` に出力 |

## スライド一覧

| ファイル | 内容 |
|---|---|
| `slides/sreake-bukai-lt.md` | Sreake 部会 LT |
